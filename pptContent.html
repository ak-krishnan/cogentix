Q2: Promotion Readiness (Problem Definition)

Objective

Build a fair, leakage-free model predicting whether an employee is promotion-ready.

Target Created

Constraints

  • NO leakage
  • NO using LastPromotionMonthsAgo directly

Speaker Notes:

Early models looked too good — because they were leaky. We fixed that.

SLIDE 11 / 35

Q2: Models Tried (and Dropped Due to Leakage)

Dropped Approaches

  • Using LastPromotionMonthsAgo as a feature
  • Creating mgr_prom_rate, mgr_avg_perf, etc.
  • Using models with AUC $\approx 1.0$ → impossible & leaky
  • Attempted SHAP/permutation on misaligned feature sets → failed

Why Dropped

These models “peeked into the future,” making them invalid for HR fairness and auditability.

Speaker Notes:

Emphasize fairness and auditability in HR contexts.

SLIDE 12 / 35

Q2: Final Valid Modeling Approach

Leakage-Free Method

  • Removed all target-derived features
  • One-hot + standardized numeric features
  • Added safe engineered features (SalaryRolePct, SalaryDeptPct)
  • Performed 5-fold cross validation
  • Added OOF Target Encoding for high-cardinality:
    • ManagerID, Department, Role
  • Added interaction features

Why This Works

✔ Fair   ✔ Non-leaky   ✔ Captures manager/role patterns without cheating

Speaker Notes:

OOF encoding prevents leakage while capturing group effects. [Image of a Predictive Modeling Architecture Diagram]

Q5: Manager-Level Aggregation

For each ManagerID:

team\_size, low\_count, pct\_low, avg\_engagement, avg\_performance, shrinkage-adjusted rate (adj\_pct\_low)

Why Shrinkage?

Adjusts for small team sizes $\rightarrow$ avoids false positives by pulling extreme values toward the global mean.

Speaker Notes:

Smaller teams need correction because rates fluctuate more.

SLIDE 30 / 35

Q5: Statistical Significance Testing

Wilson Confidence Interval

  • Computes robust CI for proportions
  • If $\text{lower\_bound} \textgreater \text{global mean} \rightarrow$ manager is significantly worse

One-sided z-test

  • Tests $H_0$: manager’s engagement $\leq$ global average
  • If $p \textless 0.05 \rightarrow$ manager significantly worse

Flagging Criteria

$\text{flag} = \text{Wilson\_low} \textgreater \text{global OR } p \textless 0.05$

Speaker Notes:

Explain test intuitively; avoids false accusations.
